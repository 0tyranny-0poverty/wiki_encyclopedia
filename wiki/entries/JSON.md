stands for **JavaScript Object Notation**.  it is an [open standard](https://en.wikipedia.org/wiki/Open_standard "Open standard")  [file format](https://en.wikipedia.org/wiki/File_format "File format"), and data interchange format, that uses [human-readable](https://en.wikipedia.org/wiki/Human-readable_medium "Human-readable medium") text to store and transmit data objects consisting of [attribute–value pairs](https://en.wikipedia.org/wiki/Attribute%E2%80%93value_pair "Attribute–value pair") and [array data types](https://en.wikipedia.org/wiki/Array_data_type "Array data type") (or any other [serializable](https://en.wikipedia.org/wiki/Serialization "Serialization") value). It is a very common data format, with a diverse range of applications.

The JSON data format is language independent.  While originally derived from [Javascript](Javascript), many modern programming languages include code to generate and parse JSON-formated data, thus adopting JSON as a [REST](REST)ful uniform interface between clients, servers, and middleware.
